遥控器¶

遥控器状态同步¶

  1. 终端反馈过来的状态，类型为1,值”0|0|0|0|0|0|0|0|0”,代表”电源|照明|大风|小风|消毒|烘干|延时|中风|蜂鸣”
  2. 电源的值有0和1,0表示关闭，1表示打开，其它的任何按钮处于打开状态时，电源键的状态都必须是打开的
  3. 照明的值有0和1,0表示关闭，1表示打开
  4. 大风的值为0和1,0表示关闭，1表示打开，大风与小风的激活状态是互斥的，也就是开大风时小风必须关闭，开小风时打开必须关闭
  5. 小风的值为0和1,0表示关闭，1表示打开，状态说明同大风
  6. 消毒的值为0和1，0表示关闭，1表示打开，消毒与烘干是互斥的
  7. 烘干的值为0和1，0表示关闭，1表示打开，消毒与烘干是互斥的
  8. 延时的值为0和1，0表示关闭，1表示打开

油烟机遥控器¶

不带集成灶的油烟机¶

  • 风速
大风、小风、中风，这3个风速是互斥的，也就是有一个打开时，另外几个必须是关闭的
  • 延时
# 延时是相对于风速而言的，就是起到延时关闭风扇的功能。
# 只有打开风速时点击延时才有效果，否则点击无反应;当开启延时时，再点击风速时就直接关闭掉风速和延时。
# 当风速和延时同时开启时，再点击延时时关闭延时，但不关闭风速。
  • 照明

带集成灶的油烟机¶

增加的按钮：消毒、烘干、自动，其中“自动”是指按下该键后，自动在消毒和烘干之间切换
